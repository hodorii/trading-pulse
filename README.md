# 📈 Trading Pulse (트레이딩 펄스)

> **Gemini CLI AI Agent가 실시간 실측 데이터를 바탕으로 발행하는 정밀 트레이딩 리포트 서비스입니다.**

## 🚀 서비스 바로가기
**[https://hodorii.github.io/trading-pulse](https://hodorii.github.io/trading-pulse)**

## 🔍 검색 및 필터 기능

본 페이지에서는 포괄적인 검색 및 필터링 기능을 제공합니다:

- **텍스트 검색**: 제목, 내용 기반 fuzzy search (Fuse.js)
- **유형 필터**: IDR, MAD, TTO, TSM, 시장, 가치, 수급, 리스크, 결정, 가이드, 통합 등
- **종목 필터**: SK하이닉스, 삼성전자, 미래에셋증권, 제룡전기, 한화, 현대차 등 주요 종목
- **날짜 필터**: 시작일~종료일 범위 검색

## 📋 주요 제공 기능
- **[IDR] Integrated Daily Routine**: 글로벌 매크로 및 섹터 심층 분석
- **[MAD] Multi-Agent Decision**: AI 에이전트 간 토론을 통한 최종 의사결정 가이드
- **[TSM] Trading Score Matrix**: 실적(EQS), 수급(SMR), 가치(VPR) 수치화 평가

## 🛠 시스템 아키텍처
- **Analysis Engine**: [trading](https://github.com/hodorii/trading) (Private)
- **Report Hosting**: GitHub Pages (Jekyll)
- **Search Engine**: Fuse.js (Client-side)
- **Static Index**: search.json (Build-time generated)
