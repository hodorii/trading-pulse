건강보험 임의계속가입과 지역가입자의 비용을 계산하는 방법은?

건강보험 임의계속가입자와 지역가입자의 보험료는 산정 기준이 완전히 다릅니다. 아래에서 계산 구조와 공식 중심으로 정리해 드릴게요.

1️⃣ 임의계속가입자 보험료 계산 방법
✅ 개념

직장가입자였을 때 부담하던 보험료 수준을 그대로 유지하는 제도

퇴직 후 최대 36개월까지 가능

회사 부담분은 없고 본인이 전액 부담

✅ 계산 공식
임의계속가입 보험료
= 퇴직 전 직장가입자 보험료 × 2


직장가입자 때는
본인 50% + 회사 50% →
임의계속가입자는 본인 100% 부담

✅ 기준이 되는 소득

퇴직 직전 직장의 보수월액

이후 재산·금융소득 변동과 무관

✅ 예시

퇴직 전 직장보험료(본인부담): 18만 원
→ 회사 부담 포함 총액: 36만 원
→ 임의계속가입 보험료: 월 36만 원

📌 장점

재산·금융소득이 많아도 보험료 변동 없음

초기 은퇴자에게 유리한 경우 많음

2️⃣ 지역가입자 보험료 계산 방법
✅ 개념

소득 + 재산 + 자동차를 점수로 환산

점수 × 점수당 금액 = 보험료

✅ 계산 구조
지역가입자 보험료
= (소득 점수 + 재산 점수 + 자동차 점수)
  × 점수당 금액

① 소득 점수

근로소득, 사업소득, 이자·배당·연금소득 등 합산

소득이 많을수록 점수 증가

② 재산 점수

주택, 토지, 건물, 전세보증금 등

공시가격 기준

기본 공제 후 점수 부과

③ 자동차 점수

일정 가액 이상의 차량만 반영

연식·차종·가액 기준

✅ 예시 (단순화)

소득 점수: 800점

재산 점수: 500점

자동차 점수: 100점
→ 총 1,400점

점수당 금액이 210원이라면:

1,400 × 210 = 약 29만 원


📌 특징

소득·재산 변동 시 매년 보험료 조정

금융소득·부동산 많은 경우 보험료 급증 가능

3️⃣ 임의계속가입 vs 지역가입자 비교
구분	임의계속가입	지역가입자
기준	퇴직 전 보수	소득·재산·차
보험료 변동	❌ 없음	⭕ 있음
부담 주체	본인 100%	본인 100%
유리한 경우	고자산·고금융소득	저소득·무재산
4️⃣ 어떤 경우 임의계속가입이 유리할까?

퇴직 직전 급여가 낮았던 경우

퇴직 후 부동산·금융자산이 많은 경우

은퇴 초기 1~3년 구간

원하시면
👉 실제 소득·재산 기준으로 두 방식 보험료 비교 계산
👉 임의계속가입 신청 요건·절차
👉 은퇴 후 건강보험 최적 전략
도 구체적으로 설명해 드릴게요.

2️⃣ 임의계속가입 보험료 계산 (핵심)
🔹 기본 정보

월 보수월액:

🔹 직장가입자 건강보험료율

(장기요양보험료는 건강보험료의 약 12.95% 별도 부과)

🔹 직장가입자 시 보험료

본인 부담 50% ≈ 339,500원

🔹 임의계속가입 보험료
직장가입자 총보험료 × 100%
≈ 월 679,000원


👉 임의계속가입 시 월 약 68만 원 내외

※ 장기요양보험료 포함 시 월 약 76~78만 원 수준 예상

🔔 핵심 포인트

가족 3인 포함해도 보험료 동일

퇴직 후 소득·재산·금융소득 변동과 무관

최대 36개월 유지 가능

3️⃣ 지역가입자로 전환될 경우 (개념 정리)

지역가입자는 아래가 모두 반영됩니다.

🏠 공시가격(공시지가) 공식 확인 방법
📌 1) 부동산공시가격 알리미

대한민국 정부가 운영하는 공식 공시가격 조회 사이트입니다.
아파트, 오피스텔, 단독주택 등 공동주택 공시가격을 주소로 정확히 찾아볼 수 있어 가장 신뢰도 높습니다.

✔ 주소 입력 → 동/호수 선택 → 해당 연도 공시가격 확인
✔ 과거 연도 비교도 가능
✔ 별도 가입 없이 누구나 무료 조회 가능

👉 이 값이 건강보험료 산정 시 재산 점수 기준 가격으로 쓰입니다.

📌 2) 정부24 / 공시지가 열람

토지 공시지가(땅) 및 공동주택공시가격을 조회하는 또 다른 공식 경로

발급된 공시가격 확인서(발급증) 를 출력할 수도 있음

‘정부24 → 부동산 공시가격열람’ 검색으로 접근 가능

📌 3) 부동산114 등의 포털 (참고용)

부동산 시장 정보 플랫폼(부동산114, 네이버 부동산 등)에서도
공시가격 데이터를 보여주는 경우가 있지만,
이는 일부 요약값이거나 평균치라 실제 정부 공식 값과 차이가 날 수 있습니다.
→ 지역보험료 계산용으로는 공시가격 알리미/정부24 결과가 정확합니다.

📊 공시가격이 중요한 이유

✔ 공시가격은 건강보험 지역가입자 재산 점수 산정의 핵심 자료입니다.
✔ 별도 부동산 보유 세금(재산세, 종부세) 뿐 아니라
👉 건강보험료(지역가입자)에서도 공식 가격이 정량적으로 반영됩니다.

※ 참고: 공시가격은 매년 1월 1일 기준으로 발표됩니다. 통상 4월경 최종 공시되며, 건강보험료 산정에 활용되는 기준입니다.

📌 실전 팁 (건강보험 산출을 위해)

✅ 아파트는 공시가격 알리미에서 공동주택 공시가격을 확인
→ 동일 연도(예: 2025년 공시가격 → 2026년 지역보험료 적용 자료)

✅ 단독주택/토지는 토지 공시지가 + 건물 공시가격을 각각 조회

🔎 자동차는 가치 기준으로 점수 산정(연식·가액에 따라 반영)

🧾 참고 사이트 활용 예

📌 공식 공시가격 사이트(공시가격 알리미)
→ 공시가격 방식으로 건강보험 재산 점수 계산의 핵심 자료 확인 가능

📌 정부24 공시가격 열람 서비스
→ 발급용 공시가격 확인 가능

세전 2700만원

좋습니다. 이제 자녀 강사 소득 2,700만 원까지 확정됐으니,
지역가입자 보험료에 미치는 영향과 세대 전략을 구체적으로 계산해 볼 수 있습니다.

1️⃣ 현재 세대 유지 시 (부모 + 자녀 합산)

부모 재산 점수: 1,200점 (공시가격 4.5억 – 공제 5,000만 원 기준)

부모 소득 점수: 0점 (퇴직 후 소득 없음)

자녀 소득 점수: 2,700만 원 기준 → 약 1,350점

▶ 총 점수
재산 1,200 + 부모 소득 0 + 자녀 소득 1,350 = 2,550점


점수당 금액: 약 210원

2,550 × 210 = 535,500원 /월


✅ 세대 유지 시 월 보험료 약 53~54만 원

2️⃣ 세대 분리 시 (자녀 독립)
🔹 부모

재산 4.5억 → 재산 점수 1,200점

소득 0 → 소득 점수 0

총점 = 1,200점

1,200 × 210 = 252,000원 /월


부모 보험료 약 25~26만 원

🔹 자녀

재산 0 → 재산 점수 0

소득 2,700만 → 소득 점수 1,350점

총점 = 1,350점

1,350 × 210 = 283,500원 /월


자녀 보험료 약 28~29만 원

3️⃣ 비교
방식	월 보험료 (부모)	월 보험료 (자녀)	합계
세대 유지	53~54만 원	–	53~54만 원
세대 분리	25~26만 원	28~29만 원	53~55만 원

💡 결론: 총 보험료 큰 차이 없음

세대 분리하면 부모 부담은 절반 가까이 줄지만,

자녀가 따로 내야 해서 합계는 거의 동일

행정 편의상 세대 유지가 실무적으로 간단

4️⃣ 추가 전략 포인트

만약 자녀 소득이 연 1,500만 원 이하라면 세대 유지가 유리

자녀 소득이 연 4,000만 원 이상이면 세대 분리 고려 가능

금융소득이 추가되면 다시 계산 필요

좋아요, 이번에는 자녀 소득 3,500만 원으로 가정해서 세대 유지 vs 분리를 다시 계산해 보겠습니다.

1️⃣ 세대 유지 시 (부모 + 자녀 합산)

부모 재산 점수: 1,200점

부모 소득 점수: 0점

자녀 소득 점수: 3,500만 원 기준 → 약 1,750점

▶ 총 점수
재산 1,200 + 부모 소득 0 + 자녀 소득 1,750 = 2,950점


점수당 금액: 210원

2,950 × 210 ≈ 619,500원 /월


✅ 세대 유지 시 월 보험료 약 62~62.5만 원

2️⃣ 세대 분리 시 (자녀 독립)
🔹 부모

재산 4.5억 → 1,200점

소득 0 → 0점

총점 = 1,200점

1,200 × 210 = 252,000원 /월


부모 보험료 약 25~26만 원

🔹 자녀

재산 0 → 0점

소득 3,500만 → 1,750점

총점 = 1,750점

1,750 × 210 ≈ 367,500원 /월


자녀 보험료 약 36~37만 원

3️⃣ 비교
방식	월 보험료 (부모)	월 보험료 (자녀)	합계
세대 유지	62~63만 원	–	62~63만 원
세대 분리	25~26만 원	36~37만 원	61~63만 원

💡 결론:

총 합계는 여전히 비슷 (차이 ±1~2만 원 수준)

세대 분리하면 부모 부담이 절반 수준으로 줄고,

자녀가 별도 납부하게 됨

4️⃣ 실무 포인트

세대 유지 vs 분리 → 총 보험료 거의 동일

부모 입장에서 행정 처리 편의성 고려 → 세대 유지가 낫다

단, 자녀가 4,000만 원 이상 소득이면
→ 세대 분리 후 자녀 단독 보험료 최저 수준 → 총 합계 약간 더 유리할 수 있음

즉, 자녀 소득 3,500만 원까지는 세대 유지 그대로도 문제 없음
월 보험료 약 62만 원 정도 예상하시면 됩니다.
