# 📍 [가이드] HPSP 내일의 쉬운 전략 (2026-02-04)

## 1. 실전 매매 전략 요약 (Trading Map)

| 핵심 지표           | 가격 / 수치  | 행동 지침                              |
| :------------------ | :----------- | :------------------------------------- |
| **최적 매수 가격**  | **45,000원** | 여기까지 내려오면 "묻지마 매수"        |
| **1차 목표가**      | **50,000원** | 도달 시 "와! 수익이다" 하고 일부 매도  |
| **최종 목표가**     | **55,000원** | 목표 달성! 전량 매도 고려              |
| **탈출(손절) 가격** | **43,500원** | 기관 형님들이 포기하는 가격, 일단 도망 |

## 2. 매수 가이드
- **언제 사나요?**: 내일 아침 장 시작하고 30분 이내에 주가가 파란색(하락)일 때 조금씩 나눠서 사세요.
- **팁**: 45,500원 밑에서 살 수 있다면 베스트입니다. 기관들이 이 근처에서 주가를 지켜줄 확률이 매우 높습니다.

## 3. 매도 가이드
- **수익은 언제 챙기나요?**: 주가가 48,000원을 돌파해서 전고점을 넘어서면 기분이 좋아지실 겁니다. 그때 절반만 팔아서 확정 수익을 챙기세요.
- **리스크 관리**: 만약 주가가 43,500원 밑으로 툭 떨어진다면, 뒤도 돌아보지 말고 일단 빠져나오는 것이 소중한 자산을 지키는 길입니다.

## 4. 내일의 행동 지침 (Action Items)
1. **아침 9시**: 삼성전자가 빨간 불로 시작하는지 확인하세요. (삼성전자가 가면 HPSP도 갑니다)
2. **오전 10시**: 기관 투자가들이 오늘도 '사자' 주문을 넣고 있는지 확인하세요.
3. **오후**: 주가가 46,000원 위에서 버티고 있다면 편안하게 소지(Hold) 하셔도 됩니다.
